🛒 Online Sales Dashboard — Power BI 

📊 Project Overview

This project presents an interactive Online Sales Dashboard built using Power BI.
The dashboard provides insights into sales performance, customer behavior, product trends, and revenue distribution across different countries.

It helps businesses analyze sales data and identify key patterns to improve decision-making and business strategies.
